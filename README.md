# Advent of Code

Solutions for puzzles from [Advent of Code](https://adventofcode.com/).

To solve puzzle pipe the input into the solution.

- Windows:
```ps1
Get-Clipboard | python $YEAR/$DAY.py
```

- Macos:
```sh
pbcopy | python $YEAR/$DAY.py
```

- Linux:
```sh
xclip | python $YEAR/$DAY.py
```
